# AI 기반 실시간 최저가 모니터링 서비스 기획안

## 1. 배경 및 문제 정의

### 1.1 시장 배경
- 온라인쇼핑은 PC 중심에서 모바일 중심 구매 환경으로 빠르게 이동 중
- Adobe Analytics 기준, 2024년 연말 쇼핑 시즌 모바일 매출 비중은 53%로 전망되었으나 2025년 실제로는 56.4%까지 상승
- 상품 탐색부터 구매 결정까지 전 과정이 점점 더 작은 화면(모바일) 안에서 이루어지는 추세

### 1.2 소비자 행동 데이터
| 지표 | 수치 |
|---|---|
| 최저가 탐색에 16분 이상 소요하는 비율 | 94% (구매 전 가격비교 시 65% / 그 외 36%) |
| 구매 전 가격비교를 하는 소비자 비율 | 83% |
| 4개 이상 사이트를 검색하는 비율 (2023 → 2024) | 20% → 24% |

- 소비자들은 가격 때문에 구매를 미루거나 포기하는 경향을 보임

## 2. 사용자 리서치

### 2.1 인터뷰 기반 도출 니즈
인터뷰 발화에서 공통적으로 나타난 페인포인트:
- 하루에도 여러 번 바뀌는 가격을 매번 확인할 시간이 없음
- 최저가인 줄 알고 구매했으나 다음 날 더 싸진 것을 보고 후회한 경험
- 원하는 목표가에 도달했을 때만 알림을 받고 싶음 (매번 직접 확인하고 싶지 않음)
- 여러 쇼핑몰(쿠팡, 네이버, 11번가 등) 탭을 오가며 비교하는 것이 피곤함
- 기존 가격비교 사이트는 불필요한 정보와 광고가 많아 오히려 혼란스러움
- 사소한 가격 변동까지 알림이 와서 결국 알림을 꺼버리게 됨
- 가격 추이 그래프가 있으면 "지금이 살 타이밍인지" 직관적으로 판단하기 좋음
- 필터가 너무 많으면 처음에 무엇을 눌러야 할지 몰라 혼란스러움
- 알림을 누르면 바로 구매까지 이어지는 편의성을 선호함

### 2.2 페르소나
**"지금 사는 게 맞나? 조금만 더 기다리면 더 싸질까?"**

- **직업**: 취업 준비생, 곧 입사를 앞두고 업무용 노트북 구매 계획 중
- **목표**: 예산 안에서 성능 좋은 노트북을 최대한 싸게 구매
- **행동 패턴**: 관심 상품 발견 시 여러 쇼핑몰 탭을 열어두고 비교하지만, 매번 다시 확인하기 번거로워 미루다가 타이밍을 놓친 경험 있음
- **페인포인트**: 하루에도 여러 번 가격이 바뀌는데 매번 확인할 시간이 없음; 최저가인 줄 알고 샀는데 다음 날 더 싸진 것을 보고 후회한 적 있음

### 2.3 인터뷰 대상자 요약
| 대상 | 특징 | 주요 니즈/페인포인트 |
|---|---|---|
| A씨 (29세, 직장인) | 뷰티/전자기기 등 관심 상품이 쌀 때만 알고 싶음, 매일 앱 켜서 확인하는 것이 귀찮음 | 알림이 너무 많거나 늦음, 여러 커뮤니티·핫딜 채널을 상시 모니터링해야 함 |
| B씨 (32세, 직장인) | 노트북·로봇청소기 같은 고가 상품은 역대 최저가 근처에서 구매하고 싶음 | "지금 사도 되나?" 판단 근거 부족, 목표가 대비 얼마나 근접했는지 알기 어려움 |

## 3. 솔루션 방향성 (기존 서비스 Good Point는 살리고 Pain Point는 개선)

| Pain Point | 개선 방향 | 분류 |
|---|---|---|
| 가격 변동을 직접 비교·판단하기 어려움 | 가격 추이 그래프로 바로 판단 가능하도록 시각화 | Trust |
| 사소한 가격 변동까지 알림이 와서 알림을 꺼버림 | 정말 필요한 가격 변동만 선별하여 알림 | Trust |
| 가격이 싼지 비싼지 스스로 판단해야 함 | "살 타이밍"을 직관적으로 판단해주는 기능 제공 | Simplicity |
| 필터가 너무 많아 처음 사용 시 혼란 | 필터는 최소화하고 필요한 항목을 자동 추천 | Simplicity |
| 여러 쇼핑몰 탭을 오가며 비교해야 하는 번거로움 | 알림을 누르면 바로 구매까지 이어지는 원스톱 경험 | Convenience |

## 4. 핵심 기능 (Key Feature)

### 4.1 크로스플랫폼 최저가 비교
- 여러 쇼핑몰의 가격을 한 화면에서 비교하여 어디가 진짜 최저가인지 표시
- 특정 쇼핑몰에 치우치지 않는 객관적인 비교 기준 제공
- 가격 추이 그래프로 지금이 살 타이밍인지 직관적으로 판단 가능

### 4.2 AI 기반 실시간 최저가 모니터링
- 등록한 상품의 가격을 주기적으로 재조회하며 실시간 추적
- 단순 가격 변동이 아닌, AI 판단 로직으로 진짜 "살 타이밍"만 필터링
- 가격 추이 그래프 기반의 직관적 구매 시점 판단 지원

### 4.3 로그인 없는 간편 등록
- 회원가입 없이 바로 관심 상품 등록 가능
- 최소한의 필터와 자동 추천으로 진입장벽 최소화
- 위시리스트를 한 화면에서 관리할 수 있는 대시보드 제공

## 5. 서비스 컨셉 요약
> **AI 기반 실시간 최저가 업데이트 서비스**
> 여러 쇼핑몰에 흩어진 가격 정보를 한곳에 모으고, AI가 "지금이 살 타이밍인지"를 판단해 알려줌으로써
> 사용자가 매번 직접 가격을 비교·확인해야 하는 번거로움과 불안감(구매 후 후회)을 해소한다.
